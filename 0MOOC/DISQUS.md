# DISQUS 私人教程

## 背景
+ win XP/win 8.1
+ gitbook webpage

## 安装
+ 点击 more下拉菜单（倒三角图标）-> find plugin
+ 查找disqus
+ 根据disqus的README提示操作
 + 在disqus.com注册用户名shortname
 + 在本书的book.json输入如下代码：
        {
            "plugin": ["disqus"]
            "pluginConfig": {
                "disqus": {
                    "shortName": "刚刚注册的shortname"
                }
            }
        }
