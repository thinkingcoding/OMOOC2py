- 极简交互式日记系统，作用如下：
  - 一次接收输入一行日记
  - 保存为本地文件
  - 再次运行系统时，打印出过往的所有日记
 
 - 使用方法：
  - 操作系统：windows
  - 下载 simpleblog.py
  - 双击打开文件
  - 在提示“Enter：”旁输入一行文字，并点击回车。
  - 程序自动退出。
  - 再次双击打开，可看到上次输入的文字已保存。再次在“Enter：”旁输入文字，并回车退出。
  - 文件保存在simpleblog.py相同的文件夹内（simpleblog.txt）。
